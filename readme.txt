Portfólio - Camila Scarpioni

Estou criando este projeto com a intenção de divulgar meus trabalhos com meu portfólio, desenvolvi a página principal e as outras páginas ainda estão em desenvolvimento.
Minha motivação para esse projeto foi justamente para me desafiar no front-end e para divulgar meus trabalhos, já que vou atualizar o site sempre, para mostrar projetos futuros e etc.
Estou desenvolvendo meu aprendizado em html e css, e começando a aprofundar mais sobre javascript, para adicionar mais interações no site.
Por enquanto, lancei somente uma versão básica, apenas para salvar meu projeto e começar a divulgá-lo, sem formatação de responsividade, mas estou trabalhando para que o site fique responsivo para qualquer tipo de computador, celular ou tablet.

Obrigada!
